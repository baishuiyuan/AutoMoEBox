AutoMind-MoE 项目完整官方描述
 
项目名称
 
AutoMind-MoE
全自动自进化轻量化混合专家模型系统
 
 
 
项目简介
 
AutoMind-MoE 是一款轻量化、本地离线、全自动自进化的混合专家（MoE）智能模型系统，采用空架构无预训练设计，从零基础开始自主学习与成长。系统内置8个基础专家模块，支持全自动学习、全自动优化、全自动训练调度、专家无限自动拆分、长期文件记忆、多模态交互、MCP协议接入，全程无需数据库、无需人工干预、无内容审查与敏感词限制，可完整复制移植到任意设备无缝继续运行，是真正可落地、可成长、可无限扩展的私有智能中枢。
 
 
 
核心理念
 
- 从零成长：空模型初始化，无预设知识、无初始权重，所有能力来自后天学习
- 完全自治：自动学习、自动优化、自动训练、自动拆分、自动记忆、自动调度
- 绝对私有：本地运行、不上云、不联网、数据完全自主掌控
- 无任何限制：无内容过滤、无专家数量上限、无输入输出约束、无使用门槛
- 开箱即用：小白可一键部署、一键运行、一键移植
- 持续进化：越使用越智能，越学习越专业
 
 
 
核心功能与特性
 
1. 空架构轻量化MoE模型
 
- 纯空白模型架构，无预训练、无内置数据、无预设行为
- 原生8个独立专家网络，搭配动态门控路由实现智能任务分配
- 轻量化结构设计，低资源占用，支持CPU与GPU自动切换运行
- 专家独立训练、独立存储、独立进化，互不干扰
 
2. 专家无限自动拆分与扩展
 
- 后台实时监控专家数据量、存储规模与调用频率
- 达到阈值自动触发精细化拆分，单个专家自动分裂为两个子专家
- 自动更新路由规则，自动分配任务与训练
- 专家数量无上限，可无限扩展、无限专业化细分
- 全程自动化执行，无需人工操作与系统重启
 
3. 全自动文件自学习
 
- 无需数据集、无需标注、无需数据库、无需人工整理
- 支持文字、对话、文档、语音、视频、网页链接直接学习
- 自动解析内容、自动提取知识、自动归档记忆、自动分配专家
- 后台静默执行，不影响前端交互与系统使用
 
4. 全自动模型自优化
 
- 自动对记忆内容去重、清洗、摘要、结构化处理
- 自动优化专家权重、路由效率与系统资源占用
- 自动清理低价值数据，保留高价值知识沉淀
- 系统持续自我优化，越使用越流畅、智能度越高
 
5. 全自动训练模式智能判断
 
- 根据内容长度、类型与用途自动选择最优训练方式
- 短文本/增量知识 → LoRA轻量微调
- 指令/对话数据 → SFT监督微调
- 偏好/对比数据 → DPO直接偏好优化
- 大量文档/知识体系 → 知识蒸馏
- 训练任务自动排队、自动调度、自动执行、自动生效
 
6. 长期文件化记忆系统
 
- 所有对话、学习内容、训练记录自动永久保存
- 纯文件目录存储，可查看、可编辑、可备份、可迁移
- 系统启动自动加载全部历史记忆，实现真正长期记忆能力
- 记忆直接用于增量微调，支持模型持续学习与知识迭代
 
7. 多模态Web交互界面
 
- 内置浏览器可视化操作界面，简单易用
- 支持文字对话、语音输入、视频解析、文档读取、网页抓取
- 实时展示专家状态、训练模式、记忆数量、拆分日志
- 全本地处理，不上云、不泄露、不联网
 
8. MCP协议标准化接入
 
- 原生支持MCP开放接口，可作为智能中枢对外提供服务
- 支持外部系统调用对话、学习、训练、专家管理等能力
- 可对接工具软件、自研系统、硬件设备、机器人、工业平台
- 本地离线运行，无云端依赖，无权限限制
 
9. 无限制自由使用
 
- 输入输出无长度、无次数、无场景限制
- 无敏感词库、无内容过滤、无输出审查、无AI安全拦截
- 无专家数量上限、无自动拆分限制、无自学习约束
- 不上云、不上报、不采集、不传输任何用户数据
 
10. 完整可移植与跨设备运行
 
- 纯绿色便携架构，无需安装、不写注册表、不绑定设备
- 整个项目目录可直接复制、压缩、迁移至任意电脑
- 移植后一键启动，模型、专家、记忆、进度完整保留
- 无缝继续进化，无需重新配置、重新训练、重新部署
 
 
 
技术栈
 
全部采用免费、开源、可商用、无限制技术构建
 
- 模型框架：PyTorch
- Web服务：Flask
- 协议支持：MCP轻量开源SDK
- 存储方案：本地文件系统（无数据库）
- 多模态解析：FFmpeg、Whisper.cpp、pdfplumber、BeautifulSoup
- 运行环境：Python 3.10–3.11
- 部署形式：绿色便携版、跨平台、离线运行
